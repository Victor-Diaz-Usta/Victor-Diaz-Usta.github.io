# Portafolio — Víctor Hugo Díaz Bautista

Este repositorio contiene mi portafolio en HTML/CSS/JS.

## Estructura
- `index.html`
- `css/styles.css`
- `js/main.js`
- `img/profile.jpg`

> Nota: En GitHub Pages, el archivo de entrada debe llamarse `index.html` (minúsculas).

## Publicar en GitHub Pages
1. Ve a **Settings → Pages**
2. En **Build and deployment** selecciona:
   - Source: **Deploy from a branch**
   - Branch: **main**
   - Folder: **/(root)**
3. Guarda y espera a que GitHub genere el enlace.

## Personalización rápida
- Cambia los links de GitHub/LinkedIn en `index.html`
- Cambia el email en la sección Contacto
- Reemplaza los proyectos por tus repos reales
- Cambia `TU_USUARIO` en las imágenes de estadísticas de GitHub
